# Tic Tac Toe

A modern Tic Tac Toe game built with Java Swing.

## Features

- **Singleplayer** - Play against AI with 3 difficulty levels (Easy, Medium, Hard)
- **Multiplayer** - Play with a friend on the same computer
- **Leaderboard** - Track wins, losses, and draws
- **Multi-language** - English and Romanian support
- **Modern UI** - Clean design with smooth animations


## How to Play

1. Launch the game and choose your game mode
2. In Singleplayer: Enter your name and select difficulty
3. In Multiplayer: Enter both player names
4. Click on the board to make your move
5. First player to get 3 in a row wins!

## Technologies

- Java 17
- Swing GUI
- MVC Architecture

## Controls

- Click cells to place your mark (X or O)
- Use settings menu to change language or audio
- View leaderboard to see top players

Enjoy the game!
